# Sonic Pi Music

Songs for [Sonic Pi](https://sonic-pi.net/) — a live coding music synthesizer.

Currently am only happy with tab3-chill-lambda.rb

## Install

```bash
brew install --cask sonic-pi
```

## Usage

Start with a song (no GUI needed — auto-reloads on save):

```bash
./play.rb tab3-chill-lambda.rb
```

Keys: `r` reload | `.` stop | `l` logs | `i` info | `e` errors | `q` quit
